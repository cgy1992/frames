Introduction

Welcome to Frames! Frames is a 2d UI library intended for game development. The design goals of Frames are simplicitly, reproducibility, and ease of development.

Frames is available under a dual-license scheme. It can be used under the GPL license or a commercial license. As of this writing, commercial licenses are free, but do require registration; please contact ben@mandible.net for details. All licenses include source access and unlimited upgrades.

Assistance is available via email (ben@mandible.net) or IRC (#frames on freenode). Commercial support contracts are available on request.

Under construction

Frames is under heavy development and this document is littered with possible additions. If your project requires a feature that is mentioned as being not-yet-available, please contact us and let us know.

Platform support

Frames officially supports Windows XP through Windows 8.1. Linux and OSX support is not yet available, but should be easy to add. Android and iOS support would require an OpenGLES renderer.

Frames can render via OpenGL (3.2 compatibility) or DirectX11 (feature level 10.0 or above). OpenGL 3.2 core and OpenGL 2.1 would be easy to add. DirectX9 and OpenGLES would be harder.

Frames runs on x86 and x64 chips. It will likely run on anything 32-bit or above that supports IEEE754 floats.

Frames does not currently run on any major commercial consoles. While we'd like to add support, we are currently unable to acquire the necessary development hardware. If your company needs this support and is able to help us acquire the hardware, please contact us.

Improving crossplatform support is a priority. If you'd like to use Frames but it is not yet available on your platform, contact us with details.

Usage

The latest source code and binary packages can be downloaded at www.framesui.com. Our binary packages contain full source code and supporting libraries, and we recommend starting from that even if you plan to modify the code yourself. Source development is done on Github.

Visit Getting Started to get Frames integrated into your game.

Visit Basics to learn the fundamentals of designing UIs in Frames.

As always, please contact us with feedback, suggestions, or complaints; if you found the documentation confusing, we want to know so we can fix it!
